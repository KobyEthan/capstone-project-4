## Run Locally

Clone the project

```bash
  git clone https://github.com/KobyEthan/jokeAPI-app.git joke-app
```

Go to the project directory

```bash
  cd joke-app
```

Install dependencies

```bash
  npm install
```
Run project

```bash
  npm start
```

Go to localhost:3000 and view the project.
